Bienvenue sur Memory

Jouez et tentez de réaliser le meilleur score possible.

Règle du jeu :

  • Retournez deux cartes à la fois.

  • Si les images sont identiques, les cartes restent visibles : vous avez trouvé une paire.

  • Si les images sont différentes, les cartes se retournent de nouveau face cachée à l’endroit où elles étaient.

  • La partie est terminée lorsque toutes les cartes ont été assemblées par paires.

Si vous voulez garder une trace de votre score, inscrivez-vous via le formulaire.

Le jeu enregistre les 5 meilleurs scores.

Pour relancer une partie, appuyez sur la barre d’espace.

Utilisation soumise aux Conditions générales et à la Déclaration de confidentialité du site.
